# Europe Market — Playwright Tests

Automated end-to-end tests for [europemarket.site](http://europemarket.site)

## 🧪 What is tested
- ✅ Homepage loads correctly
- ✅ Navigation contains all sections
- ✅ Coupon button opens the form
- ✅ Store address is visible
- ✅ Chat widget is present

## 🚀 CI/CD Pipeline
Every push to main branch triggers automatic testing via GitHub Actions.
After tests pass — website deploys automatically.

## 🛠 Tech Stack
- Playwright
- GitHub Actions
- GitHub Pages
